# 📊 Sales & Revenue Analysis Dashboard

## 📌 Overview
This project is an Excel-based dashboard built to analyze sales and revenue data.  
It provides insights into product performance, category contribution, and regional trends.

---

## 📂 Dataset
The dataset includes:
- Date
- Product
- Category
- Sales
- Quantity
- Region

---

## 📊 Dashboard Features
- Total Sales KPI
- Total Quantity KPI
- Total Products KPI
- Sales by Product Chart
- Interactive Slicers (Product, Category, Region)

---

## 🔍 Key Insights
- Mobile has the highest sales (~60,000)
- Shoes have the lowest sales (~5,000)
- Electronics category contributes the most to total sales
- North and East regions show better performance
- High-value products generate more revenue despite lower quantity

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
